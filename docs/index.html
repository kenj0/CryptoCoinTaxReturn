謝辞

本アプリケーションは、take-iwiwさんが開発した仮想通貨取引帳簿アプリをベースとし、 2017年の確定申告に限定して、各取引所の取引履歴(CSVファイル)から確定申告に必要な情報を計算するためのツールです。 コアの部分の開発はtake-iwiwさんが行っています。大変貴重なツールの公開をありがとうございます。この場を借りて御礼申し上げます。
take-iwiwさんへの寄付はこちらです。

支援
